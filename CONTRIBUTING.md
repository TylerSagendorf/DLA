# The Use of AI and LLMs is Not Permitted 

This project does not accept code or documentation generated
by LLMs or any other form of generative AI.

Additionally, pull requests or issues written with generative AI are not accepted.

[Here are some reasons to reject LLMs.](https://codeberg.org/ethical-foss/open-slopware/src/branch/main/why_not_llms.md)


# Contributions Welcome

Pull Requests and Issues are welcome.

For issues, please provide a minimal reproducible example.

Pull requests should, ideally, adhere to the existing code style and avoid introducing dependencies on other R packages.
